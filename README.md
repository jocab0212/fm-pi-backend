# FM.Pi Backend

This is the backend service for the **FM.Pi** project — a Pi-powered streaming and radio platform.  
It provides API endpoints for handling music, podcasts, and community shows.

## 🚀 Features
- Node.js + Express server
- REST API endpoints
- Ready for deployment on Render
- Environment variable support (`PORT`)

## 📦 Installation
Clone the repo and install dependencies:
```bash
git clone https://github.com/jocab0212/fm-pi-backend.git
cd fm-pi-backend
